# brainfuck-vim
A vim plugin for brainfuck programming language.

## Features
* Syntax highlight
* Interpreter

## Usage
```vim
:call brainfuck#Run()
```

## Installation
### [vim-plug](https://github.com/junegunn/vim-plug/)
```vim
Plug 'fruit-in/brainfuck-vim'
```
```vim
:PlugInstall
```

### [Vundle.vim](https://github.com/VundleVim/Vundle.vim/)
```vim
Plugin 'fruit-in/brainfuck-vim'
```
```vim
:PluginInstall
```
